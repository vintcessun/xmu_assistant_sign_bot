                API 使用文档
            

            

                

                    
1. 二维码处理

                    

                        POST /rollcall/qr
                    

                    

                        参数:
                        { "content": "string" }
                    

                    
功能: 处理二维码签到数据

                    

                        返回值: QrSignResponse 数组
                    

                

                

                    
2. 签到信息查询

                    

                        POST /rollcall/sign
                    

                    

                        参数:
                        { "user": "string" }
                    

                    
功能: 查询用户的签到状态信息

                    

                        返回值: SignResponse 数组
                    

                

                

                    
3. 自动签到

                    

                        POST /rollcall/auto_sign
                    

                    

                        参数:
                        { "user": "string" }
                    

                    
功能: 执行全课程自动签到

                    

                        返回值: AutoSignResponse 数组
                    

                

                

                    
4. 特定自动签到

                    

                        POST /rollcall/spec_sign
                    

                    

                        参数:
                        { "user": "string", "rollcall_id": "string" }
                    

                    
功能: 执行指定课程自动签到

                    

                        返回值: AutoSignResponse 数组
                    

                

                

                    
5. 推送签到

                    

                        POST /rollcall/push_sign
                    

                    

                        参数:
                        { "rollcall_id": "string" }
                    

                    
功能: 推送签到给所有用户

                    

                        返回值: PushSignResponse 数组
