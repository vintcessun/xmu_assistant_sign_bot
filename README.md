# XMUAssistantBot

本项目将核心业务逻辑与外链暴露系统高度解耦又紧密联动，解决了 OneBot 等平台难以优雅处理“富文本大内容”与“大文件”的痛点：
- **`src/logic`** 负责接收指令、调用 API、获取大文件或复杂文本内容。
- **`src/web` (Expose 系统)** 负责承接 Logic 抛出的复杂渲染任务或大文件下载，以 Web 的形式向外暴露，极大提升了用户体验。

## 🚀 核心功能与指令指南

### 1. 🌐 Web Expose 暴露系统 (`src/web`)
弥补 OneBot v11 对大文件/多文件/富文本支持的不足，基于 `axum` 提供高性能 Web 暴露服务：
- **大文件暴露 (`/file/task`)**: 
  - 基于 `ReaderStream` 实现从磁盘到浏览器的**零拷贝直传**流式下载。
  - 支持任务状态显示，外链默认 24 小时有效，彻底突破群聊文件限制。
- **富文本内容渲染 (`/md/task`)**:
  - **Markdown 转换与预览**：支持将复杂 HTML 试卷（含公式、图片、上下标、填空）转换为 Markdown，并提供**美观的网页预览**。
  - **PDF 导出**：支持将 Web 端渲染后的题库内容一键导出为 PDF 文件，方便复习与打印。

### 2. 💬 业务交互指令 (`src/logic`)
得益于自研 `#[handler]` 过程宏，所有指令逻辑均在 `src/logic` 下以即插即用的方式开发与维护。当前共支持 20 余个交互指令。

#### 🔐 登录认证指令
| **指令** | **逻辑流与优化亮点** |
|---|---|
| **`/login`** | **扫码登录**。支持双系统会话自动维护，多端不冲突；新增 LNT 失败回退 JW 机制，极大提升健壮性。底层使用 `unsafe` 字节切片进行极限性能优化。 |
| **`/logout`** | **缓存注销**。$O(1)$ 时间复杂度快速清理热数据缓存，无感状态回滚。 |
| **`/loginpwd`** | **账号密码登录**。发起账号密码登录。 |
| **`/logoutpwd`** | **密码登录注销**。删除账号密码登录数据。 |

#### 🎓 教务与学习通核心指令
| **指令** | **逻辑流与优化亮点** |
|---|---|
| **`/timetable`** | **课表渲染**。结合 LLM `ChooseTimetable` 工具精准分析学期与周数意图，调用 `ScheduleRenderer` 高效生成精美课表图片。 |
| **`/test`** | **考试列表获取**。通过 LLM 智能分析课程文本，准确、快速地抓取学习通考试/作业列表。 |
| **`/gettest`** | **试题抓取**。抓取指定考试的试题。深度联动 `web` 模块将其转化为富文本 HTML 或 PDF 外链输出。 |
| **`/testans`** | **答案抓取**。查阅指定考试/作业的正确答案与解析内容。 |
| **`/class`** | **互动列表**。查询特定课程课堂中的签到、抢答、随堂测验等互动活动列表。 |
| **`/getclass`** | **互动试题获取**。查阅指定课堂互动测验的详细问题内容。 |

#### 📍 签到与自动化指令

#### 🛠️ 实用工具与娱乐指令
| **指令** | **逻辑流与优化亮点** |
|---|---|

| **`/github`** | **开源支持**。获取机器人当前项目的 GitHub 地址。 |

### 3. 🤖 LLM 工具智能调度 (LLM Tool Calling)
不仅限于生硬的指令输入，`src/logic` 中的核心查询业务已被高度抽象为 LLM 可识别的工具 (Tool Call)。模型能够通过自然语言分析直接驱动厦大服务 API：
| **工具名称** | **功能描述** | **触发情景示例** |
|---|---|---|
| `ChooseCourse` | 智能查询/筛选/匹配教务系统中的课程信息。 | *“帮我查一下大三上学期的微积分课表。”* |
| `ChooseFiles` | 结合上下文定位并下载高数、物理等课程资料。 | *“帮我把高数最近的作业资料下载一下。”* |
| `ChooseTimetable`| 精确提取学期、周数参数用于课表查询调度。 | *“查一下大二下学期第十周的课。”* |
